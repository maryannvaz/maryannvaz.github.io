We can see that the 83 rows found earlier to be missing a value in the GRADES_ALL_G column are all also missing a value in the ENROLL column. We debated a number of approaches to fill in the missing data for these 83 rows. We decided that if these rows were clustered together in the original data (close in year), we would construct linear models for each state that predict the enrollment based on the trend in the available data. On the other hand, if the rows were spread out across the dataset, we would estimate the enrollment numbers for a state and year with missing data by taking the average of the numbers from the year before and after, or a weighted average by how far away the reference years are to the year we are estimating if the years directly next to it for the same state have a missing value as well.

Upon visual examination of the dataset, we can see that scoring data is generally collected every other year for most of the time period available, meaning that for example all 4 columns are populated for each state in 2009, but they are all empty in 2010 and so on. We decided to do a nearest neighbor fill since entires for a given state with missing data are likely to have a numeric entry for the same state in both of the surrounding years, and we are expecting this to provide a closer estimate than if we filled in the entries with the mean. We are defining the nearest neighbor based on the entry from the same state with the closest year. If there are datapoints with both earlier and later years, we decided to take a weighted average of the score from the closest year before and the score from the closest year after, weighting based on the distance of the year to the year that we are estimating in hopes of getting a more accurate estimate. We also decided to seach for values in the original dataset before any rows were removed in case it can provide a closer match based on year.

pd.set_option('mode.chained_assignment', None)

score_col_names = ['AVG_MATH_4_SCORE', 'AVG_MATH_8_SCORE', 'AVG_READING_4_SCORE', 'AVG_READING_8_SCORE']
for col in score_col_names:

    for index, row in data.iterrows():
        if np.isnan(row[col]):
            year = row['YEAR']
            state = row['STATE']

            # Find the closest years before and after the given year for the state.
            earlier_rows = original_data[(original_data['STATE'] == state) & (original_data['YEAR'] <= year) & (original_data[col].notna())]
            later_rows = original_data[(original_data['STATE'] == state) & (original_data['YEAR'] >= year) & (original_data[col].notna())]

            # If there are years for the state before this year, record the closest.
            if len(earlier_rows) > 0:
                earlier_index_closest = np.argmax(earlier_rows['YEAR'])
                before = earlier_rows.iloc[earlier_index_closest]

                # Value present before and after. Take the weighted average based on how close the years are
                # to the year we are estimating.
                if len(later_rows) > 0:
                    later_index_closest = np.argmin(later_rows['YEAR'])
                    after = later_rows.iloc[later_index_closest]         

                    diff_before = year - before['YEAR']
                    diff_after = after['YEAR'] - year

                    # Handles the potential case where the value is filled in for the original data but not
                    # since some of the rows have been removed. 
                    if diff_before == 0:
                        data[col] = before[col]
                    elif diff_after == 0:
                        data[col] = after[col]
                    else:
                        data.loc[index, col] = (diff_before * before[col] + diff_after * after[col]) / (diff_before + diff_after)

                # If there is a value for a year before but not after, use the value from the year before.
                else:
                    data.loc[index, col] = before[col]
            # If there is a value for a year after but not before, use the value from the year after.
            elif len(later_rows) > 0:
                data.loc[index, col] = after[col]
display(data)

These 10 rows are difficult to fill in using other data from Wyoming since they are all clustered together at the beginning of the time period, and there also seems to be a noticeable dropoff in values 2 years later. There are many different possibilities for what the trend could have been in this time, and it is hard to predict a general range of values. We thus decided to fill in the missing entries with the mean for the given year.

# Find the rows without a value in the pre-K enrollment column.
missing_rows = data[np.isnan(data['GRADES_PK_G'])]

# Find the mean for each year with a missing entry and use the mean in its place.
for index, row in missing_rows.iterrows():
    year = row['YEAR']
    mean = data[data['YEAR'] == year]['GRADES_PK_G'].mean()
    
    data.loc[index, 'GRADES_PK_G'] = mean

    mean_grades_all_table = mean_grades_all.to_frame()

    plt.title("Mean Grades vs. Mean Total Revenue for "+ label)
    plt.xlabel("Mean Grades")

    # Regression line to show the trend of mean revenue vs. mean grades
    fit = np.polyfit(x=table_labels["mean_grades_all"], y=table_labels["mean_total_rev"], deg=1)
